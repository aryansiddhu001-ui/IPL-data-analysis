# IPL-data-analysis
IPL Data Analysis using Python
Project Overview

This project focuses on analyzing historical Indian Premier League (IPL) match data using Python to uncover trends in match outcomes, player performances, and team behavior. The goal is to transform raw cricket data into meaningful insights using data analysis and visualization techniques.

Dataset: Dataset contains 750+ IPL matches
Includes details such as:
Match results,Teams, Player of the Match,Venue,Toss decisions, Match outcomes

Tools & Technologies:
Python, Pandas – Data cleaning and manipulation, Matplotlib & Seaborn – Data visualization, Jupyter Notebook – Analysis and reporting

Key Analysis Performed:
Cleaned and structured raw IPL dataset for analysis, Explored match results (wins, ties, no results), Analyzed Player of the Match trends, Compared team performances across seasons, Studied impact of toss decisions on match outcomes, Created visualizations (bar charts, distributions, etc.)

Key Insights:
1. Out of 750+ matches, around 743 matches had a clear result, with very few ties and no results.
2. Players like CH Gayle, AB de Villiers, MS Dhoni, RG Sharma, and DA Warner were among the most frequent Player of the Match winners.
3. Players such as Shane Watson (15 awards), Suresh Raina (14), and Gautam Gambhir (13) showed consistent high performance.
4. Most matches ended with a definite winner, indicating competitive gameplay with minimal draws.
5. Toss decisions and match conditions showed noticeable influence on match outcomes.

Visualizations
The project includes:

Bar charts for player performance, Distribution plots for match results, Comparative charts for teams and outcomes
